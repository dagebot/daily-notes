# 每日记录

一个简约的每日记录/感想博客，用 Astro 构建。

## 主题

- 简约素雅的视觉风格
- 适合阅读的长文体验
- 中文排版优化

## 技术栈

- [Astro](https://astro.build) - 静态网站框架
- 无需 JavaScript 运行时

## 本地开发

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
```

构建产物在 `dist/` 目录。

## 添加新文章

在 `src/pages/posts/` 目录下创建新的 `.astro` 文件。
